This repository contains the source code and media for my personal website.

The `main` branch is the production site, deployed at [jdecroocq.github.io](https://jdecroocq.github.io/).\
The `dev` and `content` branches are used to experiment and test new features, or to update non-coded content. These branches are deployed to addresses that are not publicly accessible.<br><br>
